# เอกสารอ้างอิง <span class="category-badge badge-reference">REFERENCE</span>

เอกสารและแนวทางที่ใช้อ้างอิงในการทำงาน

### `ref-wcag`

Checklist Accessibility — รายการที่ต้องตรวจเพื่อให้คนพิการใช้งานได้ WCAG 2.2

### `ref-aria`

วิธีทำ component ให้ screen reader อ่านได้ — ปุ่ม modal tab dropdown

### `ref-atoms`

ชิ้นส่วนเล็กสุด — Button Input Icon Badge Avatar Toggle

### `ref-molecules`

กลุ่มชิ้นส่วน — Search Bar Form Field Card Alert Dropdown

### `ref-organisms`

ส่วนใหญ่ของหน้า — Header Sidebar Data Table Modal Drawer

### `ref-templates`

โครงหน้าทั้งหน้า — Dashboard Login Settings List/Detail

### `ref-ux-writing`

แนวทางเขียน UX Copy — ปุ่ม error message placeholder ทั้งไทย-อังกฤษ

### `ref-visual`

หลักการออกแบบ — สมดุล contrast hierarchy white space

### `ref-ui-quality`

กฎตรวจคุณภาพ UI — checklist ก่อนส่งงาน

### `ref-prototyping`

ขั้นตอนทำ Prototype & User Research — ตั้งแต่คิดจนถึงทดสอบ

### `ref-design-tokens`

ค่ากลางของดีไซน์ — สี ขนาดตัวอักษร ระยะห่าง มุมโค้ง เงา

### `ref-figma-mcp`

รายการคำสั่ง Figma MCP ทั้งหมด — ใช้อ้างอิงตอนสั่ง Claude ทำงานใน Figma

### `ref-fontawesome`

รายการไอคอน FontAwesome — ค้นหาชื่อไอคอนที่ต้องการ

### `ref-hig`

แนวทางออกแบบของ Apple (HIG) และ Google (Material Design)

### `ref-skill-guide`

คู่มือสร้าง Custom Skill — วิธีเขียน โครงสร้างไฟล์ ตัวอย่าง
