# สกิลที่สร้างเอง — ดีไซน์ & QA <span class="category-badge badge-custom">CUSTOM</span>

สกิลที่สร้างเองสำหรับงานออกแบบและตรวจคุณภาพ

### `design-pipeline`

ออกแบบครบจบใน 1 คำสั่ง — ดึง Jira → วาง User Flow → ออกแบบ UI → เขียน UX Copy → ตรวจ QA

### `design-review`

ตรวจดีไซน์ครบทุกมิติ — ดูความสวย accessibility ตาม guidelines ของ Vercel

### `qa-pipeline`

ตรวจ HTML 2 ชั้น — ชั้น 1 ตรวจโค้ด (100+ กฎ) ชั้น 2 ตรวจดีไซน์ พร้อมแก้ให้อัตโนมัติ

### `design-handoff-spec`

สร้างเอกสารส่งงานให้ dev — spec ละเอียดถึงขนาดที่ dev ไม่ต้องถามอะไรเพิ่ม

### `component-state-generator`

สร้าง state ต่าง ๆ ของ component อัตโนมัติ — Default Hover Focus Disabled Error Loading

### `token-validation`

ตรวจว่าใช้สีและ spacing ตรงกับ Design Tokens ไหม — หาค่าที่ไม่อยู่ในระบบ

### `interaction-design-spec`

กำหนด animation & interaction — เช่น ปุ่มกดแล้วเด้งยังไง transition ใช้กี่ ms

### `usability-testing-plan`

วางแผนทดสอบกับผู้ใช้จริง — เขียน task scenario กำหนดกลุ่มเป้าหมาย วิธีวัดผล

### `mobile-ux-audit`

ตรวจ UX มือถือเจาะลึก — thumb zone gesture bottom nav ใช้มือเดียวได้ไหม

### `conversion-ux-review`

ตรวจ UX เพื่อเพิ่มยอดขาย — checkout ปุ่ม CTA ฟอร์มสมัคร landing page

### `clone-web`

โคลนเว็บต้นแบบเป็น HTML — ดึงสี ฟอนต์ รูปจากเว็บจริงมาทำ pixel-perfect
