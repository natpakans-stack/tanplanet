# ตัวช่วยภาษาโปรแกรม (LSP) <span class="category-badge badge-lsp">LSP</span>

Language Server Protocol — ช่วยให้ Claude เข้าใจโค้ดแต่ละภาษาได้ลึกขึ้น

### `typescript-lsp`

ช่วย Claude เข้าใจโค้ด TypeScript / JavaScript ได้ลึกขึ้น

### `pyright-lsp`

ช่วย Claude เข้าใจโค้ด Python

### `gopls-lsp`

ช่วย Claude เข้าใจโค้ด Go

### `rust-analyzer-lsp`

ช่วย Claude เข้าใจโค้ด Rust

### `kotlin-lsp`

ช่วย Claude เข้าใจโค้ด Kotlin (Android)

### `jdtls-lsp`

ช่วย Claude เข้าใจโค้ด Java

### `clangd-lsp`

ช่วย Claude เข้าใจโค้ด C / C++

### `csharp-lsp`

ช่วย Claude เข้าใจโค้ด C# (.NET)

### `ruby-lsp`

ช่วย Claude เข้าใจโค้ด Ruby

### `swift-lsp`

ช่วย Claude เข้าใจโค้ด Swift (iOS/macOS)

### `lua-lsp`

ช่วย Claude เข้าใจโค้ด Lua

### `php-lsp`

ช่วย Claude เข้าใจโค้ด PHP (Laravel WordPress)
