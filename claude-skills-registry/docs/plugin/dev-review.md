# พัฒนา & ตรวจโค้ด <span class="category-badge badge-plugin">PLUGIN</span>

สกิลสำหรับสร้างฟีเจอร์ ตรวจโค้ด และจัดการ Git

### `feature-dev`

สร้างฟีเจอร์ใหม่แบบครบวงจร — มี 3 ผู้ช่วย AI คอยวางโครงสร้าง สำรวจโค้ด และตรวจคุณภาพ

### `code-review`

ตรวจโค้ดอัตโนมัติก่อนส่งงาน — หาบั๊ก ปัญหาด้านความปลอดภัย จุดที่ปรับปรุงได้

### `pr-review-toolkit`

ตรวจ PR ละเอียดสุด — มี 6 ผู้ช่วยดูตั้งแต่โค้ด เทสต์ comment จนถึง type design

### `commit-commands`

จัดการ Git ให้เร็วขึ้น — commit สร้าง PR ลบ branch ที่ไม่ใช้แล้ว ในคำสั่งเดียว
