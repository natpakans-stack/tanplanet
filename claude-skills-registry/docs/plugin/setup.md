# ตั้งค่า & จัดการ <span class="category-badge badge-plugin">PLUGIN</span>

สกิลสำหรับตั้งค่า Claude Code และจัดการ CLAUDE.md

### `claude-code-setup`

วิเคราะห์โปรเจกต์แล้วแนะนำว่าควรติดตั้งอะไรเพิ่ม — เหมือนที่ปรึกษาตั้งค่า Claude

### `claude-md-management`

จัดการไฟล์ CLAUDE.md (ไฟล์สั่งงาน Claude) — ปรับปรุงให้ดีขึ้น แก้ไข จัดระเบียบ

### `hookify`

สร้างกฎอัตโนมัติ — เช่น ทุกครั้งที่ commit ให้ตรวจโค้ดก่อน หรือ ห้ามลบไฟล์สำคัญ
