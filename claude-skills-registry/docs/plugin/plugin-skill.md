# สร้าง Plugin & Skill <span class="category-badge badge-plugin">PLUGIN</span>

สกิลสำหรับสร้าง plugin, skill, MCP Server และ AI agent ใหม่

### `plugin-dev`

ชุดเครื่องมือสร้าง plugin ใหม่ให้ Claude — ตั้งแต่วางโครงสร้าง เขียน skill สร้าง agent ตั้ง hook

### `skill-creator`

สร้างสกิลใหม่ ปรับปรุงสกิลเดิม วัดผลว่าทำงานดีแค่ไหน แล้วเปรียบเทียบกัน

### `mcp-server-dev`

สร้าง MCP Server เอง — เชื่อมต่อ Claude กับเครื่องมือภายนอกที่ต้องการ

### `agent-sdk-dev`

สร้างแอปที่ใช้ Claude Agent SDK — ทำ AI agent ของตัวเอง
