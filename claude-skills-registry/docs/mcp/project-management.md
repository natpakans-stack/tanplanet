# จัดการโปรเจกต์ & Ticket <span class="category-badge badge-mcp">MCP</span>

MCP Server สำหรับเชื่อมต่อกับเครื่องมือจัดการโปรเจกต์

### `jira`

เชื่อม Jira — อ่าน ticket ดู sprint อัปเดตสถานะงานได้จาก Claude

### `linear`

เชื่อม Linear — สร้าง issue จัดการโปรเจกต์ ค้นหางานทั้ง workspace

### `asana`

เชื่อม Asana — ดู task ติดตามความคืบหน้า มอบหมายงาน
