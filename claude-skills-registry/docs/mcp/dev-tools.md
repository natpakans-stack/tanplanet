# เชื่อมต่อเครื่องมือ Dev <span class="category-badge badge-mcp">MCP</span>

MCP Server สำหรับเชื่อมต่อกับเครื่องมือพัฒนา

### `github`

เชื่อม GitHub — ดู repo สร้าง PR จัดการ issue review โค้ดได้จาก Claude เลย

### `gitlab`

เชื่อม GitLab — จัดการ repo merge request ดู CI/CD pipeline

### `greptile`

AI ตรวจโค้ดอัตโนมัติ — เปิด PR ปุ๊บ Greptile review ให้ปั๊บ

### `serena`

วิเคราะห์โค้ดแบบเข้าใจความหมาย — หาว่าฟังก์ชันนี้ถูกเรียกจากตรงไหนบ้าง

### `context7`

ค้นหา documentation ล่าสุดของ library ต่าง ๆ — ไม่ต้องกลัวว่าข้อมูลเก่า
