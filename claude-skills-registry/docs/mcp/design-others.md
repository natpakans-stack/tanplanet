# ดีไซน์ & อื่น ๆ <span class="category-badge badge-mcp">MCP</span>

MCP Server สำหรับงานดีไซน์และเครื่องมือเสริม

### `figma`

เชื่อม Figma — ดึงดีไซน์มาทำโค้ด จับภาพหน้าจอ อ่านข้อมูล component

### `html-to-design`

แปลง HTML กลับเป็นดีไซน์ใน Figma — เอาเว็บจริงไปใส่ Figma ได้

### `playwright`

ควบคุมเบราว์เซอร์อัตโนมัติ — ทดสอบเว็บ กรอกฟอร์ม จับภาพหน้าจอ

### `fakechat`

จำลองหน้าแชท iMessage — ไว้ทดสอบ UI แชทหรือทำ prototype
