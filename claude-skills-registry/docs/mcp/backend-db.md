# Backend & ฐานข้อมูล <span class="category-badge badge-mcp">MCP</span>

MCP Server สำหรับเชื่อมต่อกับ backend และฐานข้อมูล

### `firebase`

เชื่อม Firebase — จัดการฐานข้อมูล Firestore ระบบ login hosting storage

### `supabase`

เชื่อม Supabase — ฐานข้อมูล PostgreSQL ระบบ login storage real-time

### `laravel-boost`

เครื่องมือช่วย Laravel — สร้าง migration จัดการ route ใช้ Artisan ง่ายขึ้น
