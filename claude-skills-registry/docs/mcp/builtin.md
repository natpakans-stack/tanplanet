# มาพร้อม claude.ai (Built-in) <span class="category-badge badge-mcp">MCP</span>

MCP Server ที่มาพร้อมกับ claude.ai ใช้ได้เลยไม่ต้องติดตั้ง

### `atlassian-builtin`

Jira + Confluence ผ่าน claude.ai — อ่าน ticket ค้นหาเอกสาร comment

### `gmail-builtin`

อ่านอีเมล ค้นหา ร่างอีเมลใหม่ผ่าน Claude

### `gcal-builtin`

ดูนัด หาเวลาว่าง สร้างนัดใหม่ ตอบรับนัดผ่าน Claude

### `webflow-builtin`

จัดการเว็บ Webflow — แก้ไขหน้า element style CMS ผ่าน Claude

### `asana-builtin`

จัดการ Asana ผ่าน claude.ai — ดูโปรเจกต์ task portfolio สถานะงาน
