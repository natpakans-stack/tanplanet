# BMAD — ตรวจสอบ & เครื่องมือเสริม <span class="category-badge badge-bmad">BMAD</span>

เครื่องมือสำหรับตรวจสอบคุณภาพและจัดการเอกสาร

### `bmad-adversarial`

ตรวจแบบจับผิดจริงจัง — หาจุดอ่อน ข้อบกพร่อง สิ่งที่หลุดรอด

### `bmad-edge-case`

หา edge case — กรณีพิเศษที่อาจทำให้ระบบพัง เช่น ข้อมูลว่าง ค่าติดลบ

### `bmad-editorial-prose`

แก้ภาษาและการสื่อสาร — ทำให้เอกสารอ่านง่ายขึ้น ชัดเจนขึ้น

### `bmad-editorial-structure`

จัดโครงสร้างเอกสาร — ตัดส่วนเกิน จัดลำดับใหม่ ทำให้กระชับ

### `bmad-party-mode`

ให้ AI หลายตัวคุยกัน — จำลองการประชุมทีม แต่ละตัวมีบทบาทต่างกัน

### `bmad-help`

ถาม BMAD ว่าควรทำอะไรต่อ — วิเคราะห์สถานะปัจจุบันแล้วแนะนำ next step

### `bmad-shard-doc`

แยกเอกสารใหญ่เป็นไฟล์ย่อย — เช่น markdown ยาว ๆ แยกตาม section
