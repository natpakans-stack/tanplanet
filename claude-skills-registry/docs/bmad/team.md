# BMAD — ทีม AI จำลอง <span class="category-badge badge-bmad">BMAD</span>

จำลองทีมพัฒนาซอฟต์แวร์ด้วย AI — แต่ละตัวมีบทบาทเฉพาะ

### `bmad-master`

หัวหน้าทีม — สั่งงาน agent ตัวอื่น ๆ ทั้งหมด เหมือน project lead

### `bmad-pm`

Product Manager — วางแผนผลิตภัณฑ์ จัดลำดับความสำคัญ เขียน PRD

### `bmad-analyst`

นักวิเคราะห์ — วิจัยตลาด วิเคราะห์คู่แข่ง หา insight จากข้อมูล

### `bmad-architect`

สถาปนิกระบบ — ออกแบบโครงสร้างเทคนิค เลือก stack วาง architecture

### `bmad-ux-designer`

UX Designer — วาง user flow ออกแบบ wireframe กำหนด UX pattern

### `bmad-dev`

Developer — เขียนโค้ดตาม story implement ฟีเจอร์

### `bmad-qa`

QA Tester — ทดสอบระบบ หาบั๊ก เขียน test case

### `bmad-sm`

Scrum Master — จัดการ sprint ติดตามงาน แก้ปัญหาที่ติดขัด

### `bmad-tech-writer`

นักเขียนเทคนิค — เขียนเอกสาร API doc user guide
