# เครื่องมือที่ใช้ทำงาน <span class="category-badge badge-tool">TOOL</span>

แอปพลิเคชันและเครื่องมือที่ใช้ร่วมกับ Claude Code

### `remotion`

สร้างวิดีโอและสไลด์ด้วย React — เขียนโค้ดแทนการลาก วาง ใช้ทำ presentation แบบ programmatic

### `loftlyy`

Web app สำหรับดูแบรนด์โรงแรม — สร้างด้วย Next.js 16 + React 19 + shadcn/ui รองรับ 5 ภาษา

### `masko-code`

macOS app ช่วยเขียนโค้ด — เปิดได้ทุกที่ด้วย shortcut ค้นหาและใช้งานเร็ว

### `flow`

AI writing assistant — ปรับการเขียนให้เป็นสไตล์ของเรา ใช้ได้ทุกแอป ทั้งแชท อีเมล เอกสาร

### `chops`

macOS app สำหรับ browse, edit และจัดการ AI agent skills — รองรับ Claude Code, Cursor, Codex, Windsurf, Copilot, Aider, Amp ในที่เดียว
